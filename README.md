## Tech Stack
Backend
 * Language - [Golang](https://golang.org/)
 * Framework - [Chi](https://github.com/go-chi/chi)
 * Database - [CloudSQL](https://cloud.google.com/sql)

Frontend
* Languages - [JavaScript](https://www.w3schools.com/Js/)/[HTML](https://www.w3schools.com/html/)/[CSS](https://www.w3schools.com/Css/)
* Framework - [React JS](https://reactjs.org/)
* Library - [Bootstrap](https://getbootstrap.com/)

Deployment
* Client - [Firebase](https://firebase.google.com/products/hosting)
* Server/API - [Google App Engine](https://cloud.google.com/appengine/)
* Website - [Coterie](http://coterie-react.web.app/)
